## Weclome to Editor
Editor is an open source text editor that allows you to read, write and modify text files.

### Supported Types
- Text Files (*.txt)
- HTML Files (*.html)
- JavaScript Files (*.js)
- CSS Files (*.css)
- YAML Files (*.yml)
- XML Files (*.xml)

More file types will be supported soon.

### Download
To download & install Editor, please visit https://texteditor.to

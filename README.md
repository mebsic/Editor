### Supported Types
- Text Files (*.txt)
- HTML Files (*.html)
- CSS Files (*.css)
- JavaScript Files (*.js)
- YAML Files (*.yml)
- XML Files (*.xml)
- Markdown Files (*.md)

More file types will be supported soon.
